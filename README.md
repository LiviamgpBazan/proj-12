# Estrutura de Dados com Vetores em JavaScript

## Sobre a atividade

Na realização desta atividade foram criados programas em JavaScript utilizando vetores e estruturas de repetição.

Os exercícios realizados tiveram como objetivo:
- criar vetores;
- guardar dados;
- usar estruturas de repetição;
- manipular valores;
- praticar lógica de programação.

## Exercícios realizados

### 1. Soma dos números pares
Programa que lê 10 números inteiros, guarda os valores em um vetor e soma apenas os números pares.

### 2. Guardar números em um vetor
Programa que solicita ao usuário 7 números e guarda os valores em um vetor.

### 3. Quantidade de números repetidos
Programa que guarda 10 números em um vetor e informa quantas vezes um determinado número aparece.

### 4. Vetor sem números repetidos
Programa que cria um novo vetor eliminando os números repetidos do vetor original.

## Linguagens e ferramentas utilizadas

- JavaScript
- Visual Studio Code
- GitHub

## Objetivo

Aprimorar os conhecimentos sobre estrutura de dados utilizando vetores em JavaScript.
